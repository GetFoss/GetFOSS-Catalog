## 🤝 How to contribute
If you know a great open-source program that is not on the list:
1. Fork this repository.
2. Add the program to the appropriate category using the template: `* [**Name**](Official Site Link) — Short description. ([Review](Review Link))`
3. Submit a Pull Request.

If the software is truly FOSS, we will add it to the catalog and later publish a detailed review on [getfoss.org](https://www.getfoss.org/)!

## Password Managers
* [**Bitwarden**](https://bitwarden.com/) — Bitwarden is an open-core password manager with FOSS clients and a proprietary server. Full FOSS compliance requires self-hosting with the Vaultwarden server implementation. ([Review](https://www.getfoss.org/password-managers/bitwarden-self-hosted-password-manager-open-core/))
* [**KeePass**](https://keepass.info/) — Free, open-source offline password manager. KeePass stores credentials in a local encrypted database, ensuring full user control and freedom from cloud dependencies. ([Review](https://www.getfoss.org/password-managers/keepass-password-safe-local-first-password-management/))
* [**KeePassXC**](https://keepassxc.org/) — KeePassXC is a free and open-source password manager that stores credentials locally in an encrypted database. It offers cross-platform support, Auto-Type, browser integration, and Passkey management. ([Review](https://www.getfoss.org/password-managers/keepassxc-cross-platform-password-manager/))
* [**KeeWeb**](https://keeweb.info/) — KeeWeb is an open-source password manager compatible with KeePass databases. It runs as a web application and a desktop client across Windows, macOS, and Linux. ([Review](https://www.getfoss.org/password-managers/keeweb-cross-platform-keepass-password-manager/))
